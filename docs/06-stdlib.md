# Standard Library (Stdlib)

## Core Modules

### `core` (Auto-imported)
- `print(val)`: Outputs to stdout.
- `exit(code)`: Terminates process.
- `input()`: Reads string from stdin.

### `web` 
- `fetch(url)`: Perform GET request.
- `serve(port, handler)`: Start HTTP server.

### `file` 
- `read(path)`: Read file as string.
- `write(path, content)`: Write string to file.

### `json`
- `parse(string)`: Convert JSON string to Object.
- `stringify(obj)`: Convert Object to JSON string.
